希沃 Windows 11 镜像

简介

这是一个 Windows 11 22H2 (10.0.22621.1702) ，基于希沃官方镜像升级，适配绵阳中学的希沃一体机（第五代，主板型号 MT51），预装了部分常用软件的 GHO 镜像。

驻波自认为积攒了比较丰厚的电教经验。自小学以来给班上电脑升级系统一直是每年保留节目，故可以保证此镜像不会存在运行上的问题。但还存在些许需要还原镜像前后进行的额外操作，具体见后文说明。

error_outline 注意事项和提示

1 无需键盘鼠标也可从 U 盘启动
可先进入 Windows 设置，找到高级启动并点击重启，在 WinRE 里会有从存储介质启动的选项；如果没有，要先进入启动修复，重启并提示“无法修复你的电脑”时，点“高级选项”，才会出现“从存储介质启动”。

但需注意，希沃的两个 USB-A 接口中，有一个是直接连接到内置电脑主板的，而另一个和 Type-C 接口则是通过 USB 集线器连接到安卓系统的，插在后者的接口上有时可能无法识别启动盘。

2 两种镜像可供选择
V2 镜像提供了各分区分开打包（Partition）和整个硬盘打包（Disk）两种方式。推荐选择各分区分开打包。

选择『各分区分开打包』，可避免格式化 D、E 盘，无需转移里面的重要数据。覆写 C 盘数据后，将 D 盘文件的压缩包直接解压到 D 盘根目录即可。 99% 的情况无需再配置引导，若出现问题可再通过 Dism++ 修复。

选择『整个硬盘打包』，请备份电脑显示的 SSD 三个分区里的重要数据。 这种方式建议 Ghost 中不要修改各目标分区容量，可能引起闪退。之前的的引导问题已经修复。

3 考虑冰点还原的影响
若现在你的希沃重启后数据会被清理，那么就是打开了希沃管家里的“冰点还原”。此时重装系统，相当于强制关闭了冰点还原，在希沃集控平台上可以看到记录。如果你的学校对此很在意，请不要重装系统，否则可能会招来电教和技术人员！

重装系统后如果想重新打开冰点还原，参考此处教程。

4 可能需要重新绑定学校
有朋友说重装系统后，希沃管家不会根据硬件信息自动恢复原来的学校绑定信息。那可能需要重新绑定学校，请在重装系统前记下学校的机构码（4 位数字或者字母的代码）。

重装系统后请尽快绑定。镜像里的希沃管家版本比较旧，绑定时应该无需验证管理员密码。希沃管家自动更新后可能会要求输入管理员密码。

5 关于 Windows 和 Office 的激活方式
由于硬件上一些奇怪的问题，封装镜像时 Windows 的永久数字激活在还原后会失效。2023 年底微软关闭了永久数字激活的通道，故 V2 镜像在安装时采用了 KMS38 的激活方式。

但这已经是 2026 年了，似乎早已有了成熟的数字激活替代方案——HEU KMS Activator。该工具可同时数字激活 Windows 和 Office，不喜欢目前的 KMS38 激活方式的话可以尝试。

6 建议关掉 Windows 多指手势
有的老师用希沃白板喜欢四五指并用地滑动，容易误触整得很尴尬。打开设置 > 蓝牙和其他设备 > 触摸 > 高级手势设置，关闭多指手势。

error_outline 注意事项

1. 在没有实体键盘的情况下，建议的进入PE盘的方法：
进入设置，找到高级启动并点击重启，在WinRE里会有从媒体介质启动的选项；

2. 为处理好 UEFI 启动，安装此镜像会直接覆盖整个固态硬盘，故请检查电脑显示的前3个有效分区是否有重要数据。
mz 的电脑还内置另一个 1TB 的机械硬盘，建议将镜像放置在此硬盘的分区里。
另外，Ghost若出现闪退现象，建议还原镜像是不要修改各目标分区容量。实在不行，请使用官方提供的CGI_Plus备份还原（见下）。

3. 若完成还原后开机蓝屏提示“未连接或无法访问所需设备”，请看！非常重要！
目前发现是UEFI引导的问题。请进入自己的PE系统，打开Dism++，打开电脑硬盘系统（而非PE系统）的对话，在工具里选择“引导修复”；
修复完成后系统可以启动，但开机会出现2个“Windows 11”，请进入第一个，然后在 Windows 搜索里搜索“系统配置”(msconfig)然后删去第二个引导项。

4. 已知 WPS 内测结束以后会强制要求手动安装最新版，请看！
直接按提示安装即可。之前远志楼涉及此问题的几台已安装本镜像的设备目前已经修复此问题。

5. 若你班的电脑重启后C盘文件自动重置，那么就是打开了希沃管家里的“冰点还原”，请不要安装本系统，否则相当于关闭了冰点还原，会招来电教老师和技术人员！！

一些想法（2026 年 2 月 13 日）

今天我偶然登上 123 云盘的分享页，看到这个镜像的分享链接已经有了近 5000 次浏览，1100 多次下载，内心真的是充满了感慨，再一搜，发现必应都把这个页面挂到很靠前的位置了，但自己读了下这个不像文档也不像教程的东西，真的一言难尽，里面各种左右脑互搏……所以就完善了下。

但说实话，当初做这个镜像还是有些敷衍，这个系统是直接在希沃官方 GHO 镜像上升了级又做成 GHO，虽说避免了为驱动问题焦头烂额，但说实话，可能还是偷了些懒吧。另外就是技术缘故，Ghost 操作还是太容易了，我没什么封装 WIM 的经验，之前没弄成功过，也就放弃了。

我惊诧于镜像上次更新都快两年了，还能得到大家认可（当然我还是感觉 Win11 22H2 版更流畅，后面的更新真的是一shǐ山更比一shǐ山高）。简陋也罢，臃肿也罢，这个镜像能帮到你，那我就真的很开心了。
